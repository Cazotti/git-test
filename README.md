## Criando localmente para o Servidor.
```
git init
```

## Realizando Commit's.
```
git status
git add (*, all, README.txt, *.txt)
git commit -m "Comentário aqui."
```
## Subindo para o Repositório.
```
git push
```
## Clonando Repositórios.

```
git clone https://github.com/Cazotti/git-test.git
```
